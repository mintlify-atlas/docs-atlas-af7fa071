## Overview

This configuration uses LazyVim's default keymaps with no custom overrides in `lua/config/keymaps.lua`. All keybindings come from LazyVim's sensible defaults.

<Info>
  The leader key is set to **Space** by default in LazyVim.
</Info>

## Custom Keymaps

The `lua/config/keymaps.lua` file currently has no custom keybindings:

```lua lua/config/keymaps.lua
-- Keymaps are automatically loaded on the VeryLazy event
-- Default keymaps that are always set: https://github.com/LazyVim/LazyVim/blob/main/lua/lazyvim/config/keymaps.lua
-- Add any additional keymaps here
```

To add custom keymaps, add them to this file using:

```lua
vim.keymap.set("n", "<leader>cc", "<cmd>:echo 'Custom command'<cr>", { desc = "Custom Command" })
```

## Adding Custom Keymaps

To add your own keybindings, edit `lua/config/keymaps.lua`:

```lua
-- Normal mode keymap
vim.keymap.set("n", "<leader>cc", "<cmd>:echo 'Hello'<cr>", { desc = "Say Hello" })

-- Visual mode keymap
vim.keymap.set("v", "<leader>cx", ":s/foo/bar/g<cr>", { desc = "Replace foo with bar" })

-- Multiple modes
vim.keymap.set({ "n", "v" }, "<leader>y", '"+y', { desc = "Yank to system clipboard" })
```

<Info>
  Always include a `desc` field in keymaps - it appears in the which-key popup and makes keybindings discoverable.
</Info>
